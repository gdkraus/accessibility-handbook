Link vs. Button

Length of Link Text

Page Contents

  • The link text describes where the link will take the user
  • The text "click here" does not describe where the link will go. The description for the link follows the actual link
  • The text "click here" is unnecessary because users already know they need to click a link to follow it

The link text should be descriptive of its target location. Do not use “click here” as part of the text.
